# CommProtocols_AnggiDebia
# REST API Transaction Monitoring

## Deskripsi
Repository ini berisi hasil praktikum Communication Protocols dengan studi kasus Monitoring Transaksi menggunakan REST API.

## Tujuan
- Memahami konsep komunikasi data menggunakan REST API.
- Melakukan request HTTP menggunakan Postman.
- Menganalisis response dari server.
- Memahami struktur data JSON.
- Mendokumentasikan hasil pengujian.

## Endpoint yang Digunakan

### GET Semua Transaksi
GET /api/transactions

### GET Detail Transaksi
GET /api/transactions/1

### POST Tambah Transaksi
POST /api/transactions

## HTTP Method

| Method | Fungsi |
|----------|----------|
| GET | Mengambil data |
| POST | Menambah data baru |

## Status Code

| Code | Keterangan |
|--------|--------|
| 200 | OK |
| 201 | Created |
| 400 | Bad Request |
| 401 | Unauthorized |
| 404 | Not Found |
| 500 | Internal Server Error |

## Tools
- Postman
- REST API
